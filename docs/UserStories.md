# User Stories Document

## 001 (Registration)

As a student, I want to register an account using my UWA email and a
strong password, so that I can securely access the platform's features.

## 002 (Search)

As a user, I want to search for units by keyword, unit code, or name, so
that I can quickly find the specific unit I am interested in.

## 003 (Publishing)

As a publisher, I want to post and edit my own questions about UWA
units, so that I can get help or clarification from other students.

## 004 (Answering)

As a student who has completed or is currently taking a unit, I want to
answer questions from others, so that I can share my knowledge and help
my peers.

## 005 (Commenting)

As a user, I want to comment on existing posts, so that I can ask
follow-up questions or join the discussion.

## 006 (Favorites)

As a user, I want to mark posts as "Interested" to save them in a
favorite folder, so that I can easily find and review them later.

## 007 (Rating)

As a student, I want to rate the difficulty of units I've taken, so that
other students can make informed decisions before they enroll.

## 008 (Messaging)

As a user, I want to send direct messages to others, so that I can
discuss unit-related details privately.

## 009 (Profile/Academic History)

As a student, I want to list the units I am currently taking or have
passed (along with my CGPA), so that my profile accurately reflects my
academic background.

## 010 (Notifications)

As a publisher, I want to receive real-time notifications when someone
answers or comments on my post, so that I can respond to feedback or
stay updated on the discussion promptly.

## 011 (Reporting)

As a user, I want to report posts, answers, or comments that violate
community guidelines, so that I can help maintain a safe and respectful
environment on the platform.

## 012 (Unit Page)

As a student, I want to view a dedicated page for each unit that
aggregates all related posts, ratings, and reviews, so that I can get a
comprehensive overview of the unit's quality and requirements in one
place.
