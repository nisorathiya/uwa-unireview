# UWA Unit Helper - Product Backlog (Categorized by Role)

> [cite_start]**Project:** Checkpoint Meeting 2 [cite: 1]
> [cite_start]**Application Name:** UWA Unit Helper [cite: 2]

## Role: User (General)
*General features available to all authenticated community members.*

- [ ] [cite_start]**As a** user, **I want to** search for units by keyword, unit code, or name, **so that** I can quickly find the specific unit I am interested in. [cite: 7]
- [ ] [cite_start]**As a** user, **I want to** comment on existing posts, **so that** I can ask follow-up questions or join the discussion. [cite: 13]
- [ ] [cite_start]**As a** user, **I want to** mark posts as "Interested" to save them in a favorite folder, **so that** I can easily find and review them later. [cite: 15]
- [ ] [cite_start]**As a** user, **I want to** send direct messages to others, **so that** I can discuss unit-related details privately. [cite: 19]
- [ ] [cite_start]**As a** user, **I want to** report posts, answers, or comments that violate community guidelines, **so that** I can help maintain a safe and respectful environment on the platform. [cite: 25]

## Role: Student
*Features focused on the UWA academic experience and personal academic records.*

- [ ] [cite_start]**As a** student, **I want to** register an account using my UWA email and a strong password, **so that** I can securely access the platform's features. [cite: 5]
- [ ] [cite_start]**As a** student who has completed or is currently taking a unit, **I want to** answer questions from others, **so that** I can share my knowledge and help my peers. [cite: 11]
- [ ] [cite_start]**As a** student, **I want to** rate the difficulty of units I've taken, **so that** other students can make informed decisions before they enroll. [cite: 17]
- [ ] [cite_start]**As a** student, **I want to** list the units I am currently taking or have passed (along with my CGPA), **so that** my profile accurately reflects my academic background. [cite: 21]
- [ ] [cite_start]**As a** student, **I want to** view a dedicated page for each unit that aggregates all related posts, ratings, and reviews, **so that** I can get a comprehensive overview of the unit's quality and requirements in one place. [cite: 27]

## Role: Publisher
*Features for users actively creating primary content or seeking specific help.*

- [ ] [cite_start]**As a** publisher, **I want to** post and edit my own questions about UWA units, **so that** I can get help or clarification from other students. [cite: 9]
- [ ] [cite_start]**As a** publisher, **I want to** receive real-time notifications when someone answers or comments on my post, **so that** I can respond to feedback or stay updated on the discussion promptly. [cite: 23]
